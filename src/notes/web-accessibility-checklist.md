- Add [Skiplinks](https://webaim.org/techniques/skipnav/)
- Add vertical padding to links (for tapping)
- Add `:focus` rules
- Test keyboard accessibility
- Check Chrome **Lighthouse**
- Check Firefox **Accessibility** tab
- [Audit](https://wave.webaim.org)
