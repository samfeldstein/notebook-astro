Width and height must be equal. The flex rules keep inner content centered.

```css
.circle {
	width: 2em;
	height: 2em;
	border-radius: 50%;
	border: 0.1em solid;
	display: inline-flex;
	align-items: center;
	justify-content: center;
}
```
