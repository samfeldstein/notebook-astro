[*Homebrew World*](https://spoutinglore.blogspot.com/2018/07/homebrew-world.html) seems promising. It's a simplified version of *Dungeon World*, which is a simplified version of D&D based on the PBtA game system.

*Homebrew World* won't teach you how to play a PBtA game. For that, see the [*Dungeon World Guide*](https://drive.google.com/file/d/0B8_Fz4m5hcoiTXpTbklDOF9iUHc/view?ref=dungeon-world.com&resourcekey=0-xI_68aH1lllySOdEovKvPQ), then read the [*Dungeon World* Core Book](https://sotipro.com/DW_PDF/DungeonWorld.pdf).
