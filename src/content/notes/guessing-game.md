Try to guess something about the other person by asking questions. The questions must be personal, because the point is to learn about them. So if you're trying to guess where they're from, don't ask about the weather. Instead, ask what they did for fun growing up, or how big their graduating class was, or if they lived on a highway or a dirt road.

## Things to find out

- Where they're from.
- If they're the oldest, youngest, middle, or only child.
