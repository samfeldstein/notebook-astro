Right now I'm using [Bitwarden](https://bitwarden.com/). It's open source and works on all devices.

By default you have to unlock it every time to use it in your browser, which I thought was pretty annoying so I turned that setting off. Now it's pretty seamless.

Also turned off autofill on my browser to prevent conflicts.
